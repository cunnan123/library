JSON 配置
    JSON 是一种数据格式，并不是编程语言，在小程序中，JSON扮演的静态配置的角色
    JSON文件在小程序代码中扮演静态配置的作用，在小程序运行之前就决定了小程序一些表现，需要注意的是小程序是无法在运行过程中去动态更新JSON 配置文件从而发生对应的变化的。
    JSON的值只能是以下几种数据格式：
        数字，包含浮点数和整数
        字符串，需要包裹在双引号中
        Bool值，true 或者 false
        数组，需要包裹在方括号中 []
        对象，需要包裹在大括号中 {}
        Null
